# Common API Errors

## 400 Bad Request

Possible causes:
- Invalid JSON
- Missing required fields

## 401 Unauthorized

Possible causes:
- Invalid token
- Expired token
- Missing Authorization header

## 403 Forbidden

Possible causes:
- Insufficient permissions

## 404 Not Found

Possible causes:
- Incorrect endpoint URL

## 500 Internal Server Error

Possible causes:
- Backend/server issue
